If a function is called on an object, object.y(), then `this` within `y` refers to `object`. If called directly: y(), `this` refers to the global object - for browsers it's `window`. `this` can be set explicitly by using y.call(thisArg, argument1, argument2, ...), or with  .apply() and .bind().  For further explanation, see: http://killdream.github.com/blog/2011/10/understanding-javascript-oop/#sec-2-2
