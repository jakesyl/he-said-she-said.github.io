tongcx: XHR (XMLHttpRequest) is one of the DOM APIs it abstracts. That lets you query some web page through the HTTP protocol, thus you can use it to interact with REST APIs (which several websites provide). It also abstracts over JSON-P (JSON with Padding), which is a "dirty hack" for handling cross-domain interaction over HTTP where the domain doesn't provide a CORS alternative, or the browser does not support it.
